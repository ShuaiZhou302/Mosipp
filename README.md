## Usage

### Compile and toy example

- (cd to the folder of this repo public_mosipp/)

- mkdir build

- cd build

- cmake ..

- make

- ./test_mosipp

### Notes

- The file test_mosipp.cpp provides a toy example about how to use the code (declare graphs and costs, invoke the planner, get the result).
